ABOUT THE APP

FEATURES

  • Entries allows Users to take and upload a photo (or select from archive) of entrances from their everyday destinations.
  • Home page provides a clean and easy to read Day-view map of User activity.
  • Users can select from four different map views: Play, Work, Week, and Month view.
  • List view icon on top right allows Users to select from Life, Most Visits, and Gallery view.
  • Settings page allows Users to personalize their Entries by changing the theme of the app from updated selections.
